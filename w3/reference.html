References for HTML and CSS

HTML Tags Used

This project utilizes several HTML tags, including:

  • <header> - Defines the header section of the page.
  • <nav> - Contains the navigation links for easy access.
  • <main> - Encloses the main content of the page.
  • <article> - Represents independent content that could stand alone.
  • <footer> - Contains footer information and links.

CSS Styles Used

The CSS file includes styles for:

  • body - Sets the background color and text color.
  • a - Styles for links, including hover effects.
  • table - Defines the table styles with borders and padding.
  • figure - Styles for images and captions.
  • nav ul - Styles for the navigation menu.
